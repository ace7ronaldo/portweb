Winged

Context

This project was inspired by the project: Wild. I needed to model a  3D book cover for book and inspired by that make a physical 3D model using a 3D printer.

Making

Project: Wild was still under development at that point in time so I was quite perplexed how to model a 3D cover for it. No idea how it struck me but glad it did, I decided to give my book wings without any red bull. It had to be an amalgam of both art and animals. It was a long and tedious task modeling the wing and positioning the feathers but at the end, it was so satisfying. For the 3D printed model, I first made a few testers using the laser cutter and then finally 3D printed a bookcase with wings. That way any book can have a pair of wings.
